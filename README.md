Overview

Ancient Temple Escape is an interactive browser-based puzzle game built using HTML and Tailwind CSS. The game immerses players in a mysterious ancient temple where they must solve challenges, explore environments, and escape successfully.

Features

Puzzle-Based Gameplay – Solve different challenges to progress through the temple. 
Interactive Elements – Click-based actions and decision-making. 
Responsive Design – Works on desktop and mobile devices. 
Lightweight & Fast – No heavy frameworks, optimized performance. 

Game Objective

The player is trapped inside an ancient temple and must:

Explore different sections, 
Solve puzzles or clues, 
Make correct decisions, 
Escape successfully 

Unique Aspects

Simple yet engaging concept, 
Focus on UI/UX design using Tailwind, 
Beginner-friendly game logic
